   - Marketplace / Two-sided platform / Unit economics -> use BASELINE as base
     Key signals: marketplace, supply/demand, unit economics, operations at scale
     Use Urban Company bullets with hyperlocal acquisition and unit economics language

   - AI-native / Consulting-adjacent / CoS -> use BASELINE as base, add AI projects section

2. Always start from the version closest to the JD, then inject JD keywords
3. Never mix framing styles within a single bullet (pick one version per bullet)
4. The Accenture forecast bullet is ONLY for roles that mention forecasting/pipeline/CRM
