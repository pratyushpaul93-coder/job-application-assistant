# archive/

Deprecated code and files kept for historical reference.

Rules:

- Nothing in `archive/` is ever imported, executed, or referenced by live
  code. If you find yourself reaching in here, copy the snippet you need
  into a current script and leave the original alone.
- Subdirectories are dated (e.g. `scripts_backups_2026-05/`) so the context
  of when something was retired stays attached to the files.
- Do not edit files here. If you need to change something, restore it to
  its original location first.
- Items in this directory may be deleted permanently once enough time has
  passed that nobody remembers why they existed.
